# Analysis of Jovian Atmospheric Dynamics

Research project conducted at New Mexico Tech under Dr. Raul Morales-Juberías.

## Objective

Track the motion of an atmospheric structure in Cassini imagery and estimate dynamical properties.

## Methods

- Python
- OpenCV
- SciPy
- OpenPIV
- NumPy
- Matplotlib

Image-processing and tracking workflows were developed to extract atmospheric feature positions from time-series spacecraft observations.
